# Intelligestor Backend

Backend em FastAPI para o projeto Intelligestor, com:

- Integração com OpenAI (GPT-5.1)
- Rotas de IA para BuyBox e Produtos (Mercado Livre)
- Preparado para integração com Supabase e Mercado Livre API.

## Rodar localmente

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

Acesse: http://localhost:8000/docs
